# MegaMarket

## Инструкции

### Запуск приложения

Для запуска приложения вам необходимы `Docker` и `docker-compose`.

Сборка приложения:
    
    docker-compose build
    
Запуск приложения:

    docker-compose up

### Тестирование

Тестирование приложения:

    docker-compose -f docker-compose-test.yml up
